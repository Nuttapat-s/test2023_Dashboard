ขั้นตอนการทดสอบapp (จะมีpostman collectionแนบไปให้)

1. user จะmockมาให้3user คือ test1 test2 test3 password=1234 ถ้าต้องการเพิ่มuserให้ใช้ api [post] create user
2.ทำการ[post] loginด้วยuserที่มีในDB จะได้accessToken และ refreshTokenมา
3.นำaccessTokenไปแปะในheader ทำการยิง[post] create Dashboardก่อน -> จะได้dashboardId มา หรือหา_idได้จากขั้นตอนถัดไป
4.จากรูปในแบบทดสอบรูปแรกจะต้องใช้Api [get] view all dashboard data จะสามารถเห็นข้อมูลทั้งหมดในdashboard
5.จากรูปในแบบทดสอบรูป2 จะต้องนำ_idจาก APIข้อ4หรือ จาก dashboardId ข้อ3 โดยการส่งparamเป็นidไปที่ [Get] view dashboard detail
จะทำให้เห็นdetailทั้งหมดของข้อมูลที่จะดูและใช้ Api [post] get comment in detail page เผื่อดึงข้อมูลของcommentจากfield commentIdของ
schema dashboardมาดูได้โดยจะส่งเป็นArray Stringในการหา
6.สามารถ update statusได้ตามโจทย์ โดยการ[Get] status dropdown จะได้statusมาแล้วนำไปไปupdateที่ [Put] update dashboard status 
และidได้มาจากข้อ5.
7.ในการคอมเมนต์ให้ใช้Api [Post] comment to detail page โดยdetail คือข้อความที่จะcommentและdashboardIdที่ได้มาจาก4หรือข้อ3
เมื่อcommentแล้วสามารถตรวจสอบข้อมูลจาก API [get] view all dashboard data จะเห็นdashboardที่comment updateค่าในfield commentId
8.สามารถdelete ข้อมูลdashboardได้ด้วย Api [delete] delete dashboard

accssstoken จะมีอายุการใช้งาน30นาที และ refreshTokenจะมีอายุการใช้งาน1วัน
ถ้าaccssstokenหมดเวลา สามารถนำrefreshTokenที่ยังไม่หมดอายุไปขอtokenใหม่ได้ด้วย
API [get] get new token when refresh token not expries
